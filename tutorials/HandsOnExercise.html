PETSc Hands On

PETSc comes with a large number of example codes to illustrate usage. There three main sets are in

  • ODEs - src/ts/examples/tutorials,
  • Nonlinear problems - src/snes/examples/tutorials
  • Linear problems - src/ksp/ksp/examples/tutorials
Below, we provide a walkthrough of a few of them to demonstrate different aspects of the library.

Command-line help is also available for most of the examples through the '-help' option.

The numerical results from most of the example codes can be examined using the GLVis visualization tool. This is done to illustrate one possible way to use the linear system solution provided by hypre. In practice, users may do any number of things with the solution, including visualizing it with other packages or simply passing it along to another component of the simulation code.
