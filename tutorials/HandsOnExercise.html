PETSc Hands On

PETSc comes with a large number of example codes to illustrate usage. There three main sets are in

  • ODEs - src/ts/examples/tutorials,
  • Nonlinear problems - src/snes/examples/tutorials
  • Linear problems - src/ksp/ksp/examples/tutorials

Example 1: Nonlinear ODE arising from a time-dependent one dimensional PDE

WHAT THIS EXAMPLE DEMONSTRATES:

  • Using command line options
  • Handling a simple structured grid
  • Using the ODE integrator
  • Using call-back functions

FURTHER DETAILS:

  • Mathematical discription of the problem
  • the source code

DO THE FOLLOWING:

  • Compile src/ts/examples/tutorials/ex2.c 
          cd petsc/src/ts/examples/tutorials
      make ex2
  • Run a 16 processor example on a 132x132 grid and a 4x4 processor grid (33x33 grid on each processor) with the SMG multigrid solver 
         mpiexec -n 1 ex2 -ts_max_steps 10 -ts_monitor
     [Expected output]
    
  •       mpiexec -n 4 ex2 -ts_max_steps 10 -ts_monitor -snes_monitor -ksp_monitor
     [Expected output]
    
  •      mpiexec -n 16 ex2 -ts_max_steps 10 -ts_monitor -da_grid_x 64
     [Expected output]

DO THE FOLLOWING:

  • Compile src/ts/examples/tutorials/ex11.c 
          cd petsc/src/ts/examples/tutorials
      make ex11
  • Run shallow water flow through a tiny hybrid mesh 
          mpiexec -n 1 ex11 -f ${PETSC_DIR}/share/petsc/datafiles/meshes/sevenside.exo
      [Expected output]
  • Run shallow water flow through a small mesh with a Rosenbrock-W solver 
          mpiexec -n 1 ex11 -f ${PETSC_DIR}/share/petsc/datafiles/meshes/sevenside.exo -ts_type rosw
      [Expected output]
  • Run shallow water flow through a larger quadrilateral mesh 
          mpiexec -n 16 ex11 -f ${PETSC_DIR}/share/petsc/datafiles/meshes/sevenside-quad-15.exo
      [Expected output]
